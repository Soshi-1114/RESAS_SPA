## アプリ概要

RESAS(地域経済分析システム)API を用いたシングルページアプリケーション。
[アプリはこちらから](https://resas-spa.web.app/)

## 機能

- 都道府県一覧のチェックボックス
- 都道府県にチェックを入れると、選択された都道府県の「人口構成」を表す折れ線グラフの表示

## Setup

```
git clone https://github.com/Soshi-1114/RESAS_SPA.git
npm install
npm start
```
